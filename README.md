Remake TCC

## Estrutura
- `src/app.py`: Script principal Python para manipulação do banco SQLite
- `data/app.db`: Banco de dados SQLite gerado automaticamente

## Como executar
1. Certifique-se de ter Python instalado
2. Execute o script:
   ```powershell
   python .\src\app.py
   ```

O script irá criar o banco, inserir um usuário e exibir os dados.

## Dependências
-  Python padrão
- SQL Lite
