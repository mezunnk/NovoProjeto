# Projeto Exemplo com SQLite

## Estrutura
- `src/main.py`: Script principal Python para manipulação do banco SQLite
- `data/app.db`: Banco de dados SQLite gerado automaticamente

## Como executar
1. Certifique-se de ter Python instalado
2. Execute o script:
   ```powershell
   python .\src\main.py
   ```

O script irá criar o banco, inserir um usuário e exibir os dados.

## Dependências
- Apenas Python padrão (sqlite3 já incluso)

Adapte conforme necessário para expandir funcionalidades ou integrar com outras tecnologias.
